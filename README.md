# Ubuntu-DevelopKit

Ubuntu-12.04和Ubuntu-14.04 OK 其余版本未测试

#### 注意
不要直接拷贝setup目录下的内容

#### 操作
确保网络畅通，在普通用户模式下的命令行里面输入 定位到setup所在目录

	cp setup/   ~/Desktop/  -R
	cd ~/Desktop/setup/
	sudo sh step.sh
	
#### 安装项目
* 安装了一些依赖
* 安装并配置了tftp samba nfsroot服务 并在桌面创建了链接
* 为 root，普通用户配置vim 自动缩进，字体高亮，行号等配置
* 为 vim 增加了tags标签
